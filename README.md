# JavaScript Fundamentals & DOM Manipulation

This repository includes a beginner-friendly guide to programming fundamentals and 
web development concepts of JavaScript with examples and mini-projects.

## Covered Topics

### Fundamentals
1. Introduction & Syntax
2. Variables and Data Types
3. Control Statements (if/else, switch)
4. Loops (for, while, do-while, for...in, for...of)
5. Arrays
6. Functions (declarations, expressions, arrow functions)
7. Objects
8. DOM Manipulation

### Projects
- Mini Projects (practical applications of learned concepts)
